What is HTML?

HTML stand for HyperText Markup Language, and is the basic building block for building websites. All of the websites that you see today are built using HTML. HTML is simple to get the basics of, and you can be creating your first website today!

Getting Started

So, to get started, you need to have the following things:

  • Code editor (anything will work)
  • Web browser (you can use the one you are using now)
  • A willingness to learn (need that anyways)

After you have those, you will be good to go.
